## 🚀 Overview

NitroIDE is a fast, browser-based IDE that runs entirely on the client side.

No setup. No backend. Just code.

---

## ✨ Features

* Instant execution
* Live preview
* Monaco Editor
* Virtual file system
* CLI console
* Device preview
* Full privacy

---

## 💻 Run Locally

```bash id="run01"
git clone https://github.com/nitroideofficial/nitroide.git
cd nitroide
```

```bash id="run02"
python -m http.server 8000
```

Open: http://localhost:8000
